# TShooting（Sky Lance）

ブラウザで遊べる縦スクロール2Dシューティングゲームです。  
プレイヤー機を操作して敵を撃ち落とし、ハイスコアを目指します。

## 遊び方

### 目的
- 敵を倒してスコアを稼ぐ
- 被弾を避けて生き残る

### 操作方法
- 移動: `W` `A` `S` `D` または `↑` `←` `↓` `→`
- 攻撃: `Space`（`J` キーでも可）
- リスタート（ゲームオーバー後）: `Enter`

### ルール
- 初期HPは `3`
- 敵弾に当たる、または敵機に接触するとHPが `1` 減る
- HPが `0` になるとゲームオーバー
- 敵を1体倒すごとに `100` 点
- 時間経過で敵の出現間隔が短くなり、難易度が上がる

## 起動方法

```bash
npm install
npm run dev
```

起動後、表示されたローカルURLをブラウザで開いてプレイしてください。
